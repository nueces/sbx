# sbx webpage

Static low-fi dungeon-machine themed landing page for sbx.

Theme:

- Spirit: “small boxes for dungeon runs”
- Motivation: “explore risky code safely”
- Logo: low-fi adventurer belt pouch/toolbox with a side terminal badge

Open locally:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
